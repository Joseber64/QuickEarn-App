# 📱 QuickEarn-App
**Genera ingresos rápidos mediante micro-tareas, encuestas y ofertas desde tu celular.**

---

## 🚀 Sobre el Proyecto
QuickEarn es una plataforma diseñada para el mercado mexicano que permite a los usuarios monetizar su tiempo libre de forma segura y eficiente.

### ✨ Características Principales
* **Micro-tareas:** Validación de datos y pequeñas tareas digitales.
* **Encuestas:** Paneles personalizados según el perfil del usuario.
* **Ofertas de Afiliados:** Recompensas por probar nuevas aplicaciones.
* **Pagos Rápidos:** Integración con sistemas de retiro locales.

---

## 🛠️ Tecnologías Usadas
* **Frontend:** Android SDK (Kotlin / XML)
* **Backend:** Firebase (Auth, Realtime Database, Cloud Functions)
* **Landing Page:** HTML5, CSS3 y GitHub Pages
* **Control de Versiones:** Git & GitHub
